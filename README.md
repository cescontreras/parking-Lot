# Parking Lot
This is a parking lot simple app.

This app was made using:
  ```
  React 17 (Hooks)
  Node 14.15
  Express ^4.17.1
  PostgreSQL Server 12
  Sequelize ^6.3.5
  ```

## Installation 

First you will need to install PostgreSQL server and create a database named ``parkinglot``

Then on root dir :
```
$ cd server
$ npm install
$ npm start
```
and 
```
$ cd client
$ npm install
$ npm start
```
